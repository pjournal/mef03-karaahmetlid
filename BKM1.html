BKM - Analysis

Mina Silahtaroglu

6 March 2019

In this document, we parse BKM website for monthly transactions by transaction type and analyze the data for December 2018.

Getting raw data from BKM Webpage

raw_tables <- read_html("https://bkm.com.tr/secilen-aya-ait-sektorel-gelisim/?filter_year=2018&filter_month=12&List=Listele") %>% html_table(fill=TRUE)

Processing raw data

raw_df <- raw_tables[[4]] %>% tbl_df

colnames(raw_df) <- c("type", "txn_cc", "txn_dc", "tl_cc", "tl_dc")

cards_df <- raw_df %>% slice(-(1:2)) %>% filter(type != "TOPLAM") %>%
  mutate_at(vars(txn_cc:tl_dc),funs(gsub("\\.","",.))) %>% 
  mutate_at(vars(txn_cc:tl_dc),funs(as.numeric(gsub(",",".",.)))) %>% 
  tbl_df()

The share of car rentals as the transaction count of credit cards and debit cards in the total transactions

car_rental_txn <-
cards_df %>%
  mutate(txn_total=txn_dc+txn_cc, type2= ifelse(type=="ARABA KÄ°RALAMA", "ARABA KÄ°RALAMA" , "DÄ°ÄžER")) %>%
  group_by(type2) %>%
  summarise(txn_total=sum(txn_total)) %>%
  mutate(per= txn_total/sum(txn_total))
ggplot(data = car_rental_txn, aes(x = " ", y = txn_total, fill = type2)) + geom_bar(width = 1, stat = "identity") +
  theme(legend.position="bottom", axis.title.x=element_blank(), axis.text.x=element_blank(),axis.ticks.x=element_blank()) + coord_polar("y", start=0) + 
  geom_text(aes(x=1, y = cumsum(txn_total) - txn_total/2, label=scales::percent(per)))
